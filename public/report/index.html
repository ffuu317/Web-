验收检查清单

本页用于辅助夏涵欣整理实验报告。报告正文由夏涵欣负责，本页只保留按老师验收项目准备材料的检查清单。

评分表对应材料

验收项目 需要保留的材料
项目与代码库配置 仓库首页截图、目录结构截图、hugo.toml 配置、Git 提交记录
Web 运行环境 Hugo 版本、hugo server -D 运行截图、本地页面访问截图
Linux 环境配置 Nginx 安装、系统环境、端口、防火墙配置截图
Linux 账户配置 用户目录、站点目录权限、文件属主或权限配置截图
远程管理配置 SSH 登录、远程上传、服务器目录检查截图
Web 站点部署 可访问站点、Nginx 配置、public/ 上传结果、访问验证截图

提交前检查

  • 本地是否能打开 http://localhost:1313/
  • 首页、实验记录、任务问题、部署、验收清单是否都能打开
  • /css/custom.css 是否能访问，样式是否生效
  • public/index.html 是否重新生成
  • 图片是否放在 static/images/，页面路径是否正确
  • GitHub 仓库是否包含最新的内容和静态输出
