实验报告

本页用于整理实验报告的结构与素材清单（建议由负责报告的同学维护）。

报告建议结构

  1. 实验目标与要求
  2. 环境搭建（工具安装、配置）
  3. 站点搭建过程（生成器选择、主题配置、目录结构）
  4. 页面与样式设计（配色、布局、响应式）
  5. 部署与访问（Nginx 或 GitHub Pages）
  6. 遇到的问题与解决方案
  7. 分工与协作记录
  8. 总结与收获

素材清单（待补充）

  • 关键步骤截图（安装/运行/页面效果/提交记录）
  • 关键命令与配置片段（Hugo、Git、Nginx 等）
  • 站点链接（如已部署）
